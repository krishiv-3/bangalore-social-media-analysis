# Bangalore Social Media Analysis

## Overview

This project analyzes social media profile data from Bangalore-based creators, developers, startup founders, communities, and influencers.

The goal of the project is to extract insights from profile information such as:

* Follower count analysis
* Following patterns
* Category distribution
* Profile classification
* Creator and community trends
* Bangalore tech ecosystem observations

## Dataset

The dataset contains profile information including:

* Username
* Number of posts
* Followers
* Following
* Profile category
* Bio information

Examples include:

* Software Engineers
* Tech Creators
* Startup Founders
* Data Scientists
* Developer Communities
* Media Pages
